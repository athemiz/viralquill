Secao 05 ‚Äî Problemas e Solucoes

10 Bugs, 10 Fixes

Cada problema encontrado ensinou algo sobre como orquestrar agentes autonomos. Aqui esta o timeline completo.

Loop Infinito de Mensagens Critico
Agentes entravam em loops infinitos mandando mensagens uns aos outros sem parar. Implementado Protocolo HALT com flag halted:true em project-state.json.
@mentions Nao Ativavam Notificacoes Bug
Bots postavam texto puro sem <@USER_ID>. Mensagens eram ignoradas. Fix: regra obrigatoria de mentions com tabela de IDs nos TOOLS.md.
Channel IDs vs Nomes Bug
Agentes usavam #otavio-tasks em vez de C0ADSBW4V6Y. Erro "Unknown channel" em todas as mensagens.
CRLF vs LF no Docker Bug
entrypoint.sh com line endings Windows (\r\n) causava exec: no such file or directory em container Linux. O shebang virava #!/bin/bash\r.
docker compose restart vs up --build Aprendizado
restart nao re-le o .env. GITHUB_PAT ficou ausente nos containers por horas ate descobrir que precisa de up -d --build.
cursor-small: Context Window 8192 < 16000 Bug
OpenClaw exige minimo 16000 tokens de contexto. cursor-small tem 8192. Todos os agentes falharam simultaneamente. Fix: migrar para claude-4.5-haiku (200k).
Brave Search API Rate Limit (429) Blocker
Researcher consumiu o limite rapidamente. Tentativa de criar nova conta falhou ‚Äî ate o plano free exige cartao de credito.
Proxy Offline = Sistema Morto SPOF
Quando o proxy cursor-openai cai, todos os agentes perdem acesso ao LLM. Respondiam "Connection error" no Slack. Ponto unico de falha.
Agentes "Esquecem" Instrucoes Limitacao LLM
Por compactacao de sessao, agentes esqueciam regras dos workspace files. QA chamou read sem path, Researcher disse "nao tenho tools".
Git Conflicts no Coder Resolvido
Coder criou commits locais sem remote. Necessario rebase manual, resolucao de conflitos e force push para sincronizar com origin/main.
Secao 06 ‚Äî Veredito

O Que Funcionou vs O Que Nao

‚úì Funcionou

  • Arquitetura "Team-of-Teams" e viavel ‚Äî agentes coordenam via Slack como interface assincrona
  • Coder produziu codigo real e funcional: 13,700 linhas, 139 testes, deploy no Vercel
  • Heartbeats como motor de proatividade ‚Äî agentes agem sem ser perguntados
  • Docker isolamento garante independencia de estado entre agentes
  • Custo zero via Cursor subscription ‚Äî 150M tokens que custariam ~$673

‚úï Nao Funcionou

  • Autonomia real e muito limitada ‚Äî supervisao humana constante necessaria (~15-20 intervencoes)
  • Custo de manutencao alto ‚Äî cada bug exigia rebuild de 4 containers e horas de debug
  • SPOF no proxy local ‚Äî se o PC desliga, todo o time para
  • Context window e gargalo ‚Äî modelos menores nao comportam as instrucoes
  • Loop de mensagens sem freio natural ‚Äî sem HALT, tokens queimam infinitamente
